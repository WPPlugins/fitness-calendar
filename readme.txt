== Description ==

This plugin specially made for 3 months workout program "https://www.youtube.com/embed/9NG0H1wKk7Y". you can add 7 extra videos with reps and sets with one main video.

You add week title with video title and subtitle and equipments needed for exercise .
you can add rest time between exercise too.with 7 featured videos you have the options to add exercise videos with title reps and sets .

just add [calendar] shortcode to your page and you can see the calendar on your page.

== Installation ==

This section describes how to install the plugin and get it working.

1. Download and install the plugin from WordPress dashboard.
 You can also upload the entire plugin folder to the /wp-content/plugins/ directory.
 
2. Activate the plugin through the Plugins menu in WordPress.

3. Now you can see  plugin menu in your dashboard
